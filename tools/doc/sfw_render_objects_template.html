**SFW Render Objects External Module Docs** Render Objects External Module for SFW https://github.com/Relintai/sfw Will work with any sfw version that contains the render core module. COMPILE ==================================================================================== Copy the module next to the sfw or sfwl library, and just add it to the build like any other h + cpp files. It has no dependencies.
